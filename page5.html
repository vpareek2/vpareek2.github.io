Developed a dynamic chatbot using my own Generative Pre-trained Transformer, fine-tuned with diverse therapy datasets. This system intelligently selects from specialized models based on user queries. Leveraging the power of the National Center for Supercomputing, I integrated advanced reinforcement learning techniques. Blazing Formula 1

Leveraged the Kaggle 1950-2022 F1 Dataset and Python to perform data cleansing on multiple dimensions, such as Races, Drivers, and Performance, etc., enriched with weather data scraped from Wikipedia for each race.

This was followed by the application and optimization of various ML algorithms, such as Neural Networks, Random Forest, and regressions, to predict the 2023 F1 season. Visualizations created using R and dashboards developed via Apache Superset.
